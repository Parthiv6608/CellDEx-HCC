# Cell Death Explorer (CellDEx)

An image-based machine learning pipeline for analyzing cellular states and cell death pathways using microscopy TIFF datasets.

## Goals
- Load and preprocess microscopy TIFF datasets
- Parse associated metadata
- Explore CellDeathPred-style workflows
- Benchmark supervised ML approaches for cellular state classification
- Investigate apoptosis, ferroptosis, senescence, and related stress responses
